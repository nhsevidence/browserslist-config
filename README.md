## Installation

With Yarn:

```sh
yarn add @nice-digital/browserslist-config -D
```

Or with npm:

```sh
npm install @nice-digital/browserslist-config --save-dev
```

## Usage

Add this to your *package.json* file:

```json
"browserslist": [
    "extends @nice-digital/browserslist-config"
]
```
